MV_XML_UpdateNodeFeature

Update the node.

API Definition

int MV_XML_UpdateNodeFeature(
  void                          *handle,
  enum MV_XML_InterfaceType     enType,
  void                          *pstFeature
)

Parameters

handle

[IN] Device handle, which is returned by MV_CC_CreateHandle or MV_CC_CreateHandleWithoutLog.

enType

[IN] The node type. See MV_XML_InterfaceType for details.

pstFeature

[OUT] The current node feature structure. See MV_XML_FEATURE_x for details.

Return Value

Return MV_OK(0) on success, and return Error Code on failure.
