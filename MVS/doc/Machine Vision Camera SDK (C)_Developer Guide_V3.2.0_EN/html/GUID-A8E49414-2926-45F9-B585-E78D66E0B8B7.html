Update History

The update history shows the summary of changes in MvCameraSDK with different versions.

Summary of Changes in Version 3.2.0_Aug., 2021

Version

Content

Version 3.2.0_Aug., 2021

1. The new version uses static linking library of GenICam. And the GenICam dynamic library is not provided.

2. The GigE driver supports the ARM structure and two types of ARM boards: NVIDIA TX2 and ODROID-XU4.

3. Provided the SDK installation package (Runtime package), which includes the script of driver installation and the script of enabling read and write caching for USB3 vision cameras. The provided installation packages vary with the system (ARM or x86).

4. Updated the driver name to gevfilter.

5. Added one API for clearing the streaming data buffer: MV_CC_ClearImageBuffer.

6. Added one API for rotating images: MV_CC_RotateImage.

7. Added one API for flipping images: MV_CC_FlipImage.

8. Added one API for setting gamma parameters of Bayer pattern: MV_CC_SetBayerGammaParam.

9. Added one API for decoding the lossless compression stream into raw data: MV_CC_HB_Decode.

10. Added one API for setting the packet size of USB3 vision device: MV_USB_SetTransferSize.

11. Added one API for getting the packet size of USB3 vision device: MV_USB_GetTransferSize.

12. Added one API for setting the number of transmission channels for USB3 vision device: MV_USB_SetTransferWays.

13. Added one API for getting the number of transmission channels for USB3 vision device: MV_USB_GetTransferWays.

14. Added one API for getting the GVCP command timeout: MV_GIGE_GetGvcpTimeout.

15. Added one API for setting GVCP command retransmission times: MV_GIGE_SetRetryGvcpTimes.

16. Added one API for getting GVCP command retransmission times: MV_GIGE_GetRetryGvcpTimes.

Summary of Changes in Version 3.1.0_Sep., 2019

Version

Content

Version 3.1.0_Sep., 2019

1. Added the drive support for connecting GigE camera.

2. Added API for sending PTP (Precision Time Protocol) command of taking photo: MV_GIGE_IssueActionCommand.

3. Added API for checking if device is connected: MV_CC_IsDeviceConnected.

4. Added API for clearing GenICam node cache: MV_CC_InvalidateNodes.
